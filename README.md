# Diyaa (ضياء) 🌙

A beautifully designed, location-aware Islamic prayer and Zikr application with a fully gamified experience. 

Diyaa was 100% **vibe-coded** from the ground up! By combining cutting-edge AI orchestration with pure creative energy, every line of code, UI component, and feature in this app was generated by riding the vibes—no traditional hand-coding required. ✨

## 🌟 Features

- **Context-Aware Azkar:** Dynamically filters and displays Morning, Evening, Post-Prayer, Waking up, and Sleep azkar based on the exact time of day and your current local prayer times.
- **Global Prayer Times:** Automatic, location-based prayer time calculations anywhere in the world.
- **Gamified Progression (Duolingo-style):** 
  - **Streaks & XP:** Build daily habits, track your streaks, and earn gems for completing sessions.
  - **Achievements:** Unlock badges like *Early Riser*, *The Fortress*, and *Gem Collector*.
  - **Celebrations:** Enjoy confetti animations and haptic feedback upon completing your daily goals.
- **Rewards Shop:** Spend your hard-earned gems to unlock gorgeous app themes (e.g., *Desert Dunes*, *Medina Night*) and custom notification sounds.
- **Full Localization:** Seamlessly switch between English (LTR) and Arabic (RTL) across the entire app.
- **Adaptive UI:** Stunning Light and Dark mode interfaces featuring modern glassmorphism, fluid animations, and beautiful Islamic geometry.

## 🛠️ Technology Stack
- **Framework:** Flutter
- **Language:** Dart
- **State Management:** Provider
- **Local Storage:** SharedPreferences
- **Core Packages:** `geolocator`, `adhan_dart`, `confetti`, `google_fonts`

## 🚀 Getting Started

Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your system.

1. Clone the repository
2. Navigate to the `app` directory:
   ```bash
   cd app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
